Operon is a macOS desktop application that unifies multiple AI coding agents into a single, polished interface. Switch between Claude Code, Codex, Gemini CLI, and OpenCode within the same workspace — no context switching, no separate terminals.

## Highlights

- **Multi-Agent Chat** — Talk to Claude Code, Codex, Gemini CLI, OpenCode, or custom agents in one place, and switch mid-conversation
- **Canvas Workflows** — Build visual DAG pipelines that chain AI steps together with full parallelism
- **External Agent Delegation** — Let one agent delegate tasks to others automatically across tabs
- **Skills System** — Pluggable instruction packs that give your AI specialized knowledge
- **Long-Term Memory** — Local vector + keyword hybrid search so the AI remembers your preferences across sessions
- **MCP Servers** — Connect any Model Context Protocol server (stdio / HTTP / SSE) to extend agent capabilities
- **Cron Jobs** — Schedule recurring AI tasks or workflow executions on a timer
- **Integrated Terminal** — Full PTY terminal right inside the app
- **Auto-Update** — Built-in updater keeps you on the latest version
